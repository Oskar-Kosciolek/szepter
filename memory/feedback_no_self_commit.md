---
name: Nie commituj samodzielnie
description: Użytkownik sam commituje — Claude nigdy nie uruchamia git commit ani git add
type: feedback
---

Nie wykonuj git commit, git add ani git push samodzielnie.
Proponuj tylko commit message i czekaj — użytkownik sam decyduje kiedy i co commituje.

**Why:** Użytkownik chce mieć pełną kontrolę nad historią git.
**How to apply:** Po zakończeniu zadania podaj COMMIT: "..." i zatrzymaj się. Nie uruchamiaj żadnych komend git.
